# QuickAid-First_Aid_Advisor_ChatBot
First Aid Advisor Chatbot is a web-based chatbot that provides quick and easy first-aid guidance for various medical emergencies. It uses a user-friendly interface to offer step-by-step instructions, helping users handle injuries and health issues effectively.
<br>
---
<img src="https://github.com/Eshita-Badhe/QuickAid-First_Aid_Advisor_ChatBot/blob/main/images/intr.png" alt="Chatbot Interface" style="width:50%;height:auto;">

## Features
<ul>
    <li><strong>User-Friendly Interface:</strong> Simple and intuitive design for easy interaction.</li>
    <li><strong>Hands-free interaction:</strong> Integrating speech recognition</li>
    <li><strong>Quick First Aid Tips:</strong> Provides immediate responses for common health issues.</li>
    <li><strong>Emergency Instructions:</strong> Guidance for handling critical situations like heart attacks, fractures, burns, and bleeding.</li>
    <li><strong>Symptom Checker:</strong> Helps users identify potential health problems based on described symptoms.</li>
    <li><strong>Personalized Advice:</strong> Recommendations based on age, medical conditions, or specific scenarios.</li>
    <li><strong>Responsive Design:</strong> Optimized for various devices, including desktops, tablets, and mobile phones.</li>
</ul>

## Technologies Used
<ul>
    <li><strong>Frontend:</strong> HTML, CSS, JavaScript</li>
    <li><strong>Backend:</strong> Node.js (or other server-side frameworks if applicable)</li>
    <li><strong>Database:</strong> (Optional for storing user data or logs, e.g., MongoDB, SQLite)</li>
    <li><strong>AI/Logic Engine:</strong> Rule-based decision-making or natural language processing (NLP) for dynamic responses</li>
</ul>

## Installation
<p><strong>Note:</strong> This chatbot requires a MySQL database to function properly. Ensure MySQL is installed and running on your system. Use the provided SQL queries in the <code>database.txt</code> file to set up the required tables and data.</p>

### Prerequisites
<ul>
    <li>Node.js installed on your system (for backend)</li>
    <li>Any code editor (such as VS Code)</li>
</ul>

### Steps
<ol>
    <li>Clone the repository:
        <pre><code>git clone https://github.com/your-repository/first-aid-advice-chatbot.git</code></pre>
    </li>
    <li>Navigate to the project directory:
        <pre><code>cd first-aid-advice-chatbot</code></pre>
    </li>
    <li>Install dependencies (if applicable):
        <pre><code>npm install</code></pre>
    </li>
    <li>Start the server:
        <pre><code>npm start</code></pre>
    </li>
    <li>Open your browser and visit:
        <pre><code>http://localhost:3000</code></pre>
    </li>
</ol>

## Usage
<ol>
    <li>Open the chatbot interface.</li>
    <li>Enter your query or symptoms (e.g., "How do I treat a burn?").</li>
    <li>Follow the provided instructions or safety tips.</li>
    <li>If necessary, use the symptom checker for a more detailed response.</li>
    <li>In case of severe emergencies, the chatbot will recommend contacting professional medical services.</li>
</ol>
